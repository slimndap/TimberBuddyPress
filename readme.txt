Create BuddyPress websites using Twig templates.

== Description ==

Make the Timber plugin work with BuddyPress. Create BuddyPress themes with the power of Twig templates and all other goodies that come with Timber.

Requires Timber v x.x.x.

== Installation ==

1. Install [Timber](http://wordpress.org/plugins/timber-library/)
1. Install [BuddyPress](http://wordpress.org/plugins/buddypress/)
1. Install BuddyPress for Timber (this plugin).
1. [Get started](https://github.com/jarednova/timber/wiki/getting-started) with Timber.
